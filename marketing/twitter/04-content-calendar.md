# 04 — Content Calendar (First 4 Weeks)

Calendar designed around the agreed **low-volume cadence: 2-3 organic posts/week + ad-hoc replies.**

Daily plans would be wasteful at this cadence — weekly themes work better. Each week has 2-3 anchor posts. Replies fill in around them.

---

## Cadence rules

- Max 3 organic posts/week. More than that = burnout + quality dip.
- No reply quota. But minimum 5 substantive replies/week target. Quality > count.
- Never two manifesto posts in the same week (audience fatigue).
- Sat/Sun: no scheduled posts. Organic engagement only.
- After every post: 48-hour engagement window — check + reply to repliers.

---

## Week 1 — Foundation (no broad posting, just setup)

| Day | Action |
|-----|--------|
| Mon | Update Twitter bio to "structure, context, and memory for agentic coding." Link to frame.cool. |
| Tue | Build private list of target accounts (see `06-target-accounts.md`). Set up TweetDeck columns. |
| Wed | Write + ship pinned thread (Pillar C #1 — "Why I built Frame instead of using Cursor"). 5 tweets, no images required for v1. |
| Thu | Reply-only day. Pick 3-5 high-signal tweets from the list. Reply meaningfully (Pillar B framework). No Frame mention unless organic fit. |
| Fri | Reply-only day, same as Thu. |
| Sat-Sun | No posting. Observe what got engagement. |

**Goal of Week 1:** Foundation laid. Bio + pinned + list + 5+ replies in the wild.

---

## Posts NOT scheduled in first month (and why)

- **Loom videos:** Effort > current audience size. Defer to Week 5+ when there's actual eyeballs.
- **Architecture diagrams:** Too technical for Twitter audience. Save for blog.
- **Direct comparison meme posts (Frame vs Cursor):** Adversarial tone risks backlash. Pass.
- **Star milestone post:** Triggered, not scheduled. Will go up when stars cross 500/750/1K.

---

## Post-Week-4 review

After Week 4, do a 30-min retrospective and update `decisions-log.md` with:
1. Which post(s) got >2× the engagement of average?
2. Which post(s) underperformed?
3. Which pillar drove the most profile visits / follows?
4. Are any angles wrong for Frame's actual audience?
5. Adjust calendar for Weeks 5-8 based on signal.

If a manifesto post landed hard → lean into more manifestos.
If setup porn dominated → make the visual arsenal investment first.
If replies converted better than posts → defer organic content, double down on reply game.

---

## Backup queue (if a planned post falls through)

When you can't write the planned post:
- Pillar A #5 (CLAUDE.md framing) — low effort, single tweet
- Pillar D #1 (Cursor positioning) — single tweet, evergreen
- Pillar B #3 quote (rate-limited + multi-AI) — even works as standalone insight

Never post nothing. Better a backup than a gap that breaks the rhythm.
