# Privacy

Frame is a developer tool. We respect your privacy and try to be transparent about what we collect.

## What we collect

When telemetry is enabled (the default — opt-out via Settings), Frame sends a single anonymous event each time the app is launched. The event contains:

- **App version** — e.g. `2.0.5`
- **Operating system** — e.g. `macOS`, `Windows`, `Linux`
- **An anonymous launch count** — generated by [Aptabase](https://aptabase.com), the analytics provider, with no personal identifier

This is enough for us to understand how many people are using Frame, retention over time, and which platforms / versions are in use.

## What we do **not** collect

- ❌ File paths
- ❌ File contents or code
- ❌ Project names
- ❌ AI prompts or responses
- ❌ Terminal input or output
- ❌ Email addresses or any personally identifiable information
- ❌ IP address (Aptabase strips this before storage)

## How to opt out

Open **Settings** (`Cmd+,` on macOS, `Ctrl+,` on Windows/Linux) and toggle **"Send anonymous usage stats"** off. The setting takes effect immediately and persists across launches.

You can also dismiss the one-time notice banner at the top of the app — that doesn't change the setting itself, only hides the banner.

## Where the data goes

Telemetry is sent to [Aptabase](https://aptabase.com), an open-source, privacy-focused analytics service. They do not retain IP addresses and do not share data with third parties.

## Crash reporting

Frame does **not** currently include a crash reporter. If we add one in the future, it will be opt-in and disclosed here.
